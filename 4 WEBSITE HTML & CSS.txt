theory -  

Web Development is broadly divided into two parts:

Front-end (Client-side): What the user sees and interacts with — built using HTML, CSS, and JavaScript.

Back-end (Server-side): What happens behind the scenes — handled by languages like Java, PHP, Python, and databases.

HTML (HyperText Markup Language)

HTML is the foundation of every website. It defines the structure and layout of a webpage using various tags and elements.
Examples:
<h1>–<h6> for headings, <p> for paragraphs, <a> for links, <ul>/<ol> for lists, <div> and <section> for grouping content.
HTML is not a programming language; it’s a markup language that tells the browser how to display content.

CSS (Cascading Style Sheets)

CSS adds style and design to HTML pages. It controls fonts, colors, layouts, spacing, and responsiveness.
It follows the principle of separation of concerns — HTML for structure, CSS for style.
Ways to apply CSS:

Inline CSS – inside an element tag

Internal CSS – inside <style> in the head section

External CSS – in a separate .css file linked using <link>

HTML + CSS Together

HTML provides the content and structure, while CSS defines the look and feel.
Together, tSShey create visually appealing, responsive, and easy-to-maintain websites.
